Cheatsheet Box Model

This is to help you with the Box Model while learning about CSS

The model below the table shows a visual of the Box Model

Attribute Tag Description
W & H Width & Height The width & height of the content area
Padding Padding The amount of space between the content area and the border
Border Border The thickness and style of the border surrounding the content area and padding
Border Radius Border-Radius Modify the corners of an element's border box
Overflow Overflow Controls what happens to content thats spills outside the box
Visibility Visibility Hides elements from view
Margin Margin The amount of space between the border and the outside edge of the element
